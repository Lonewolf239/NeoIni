## Human‑editable INI mode (1.7.2+)

Версия **1.7.2** добавляет **HumanMode**, предназначенный для тулзов и редакторов, в которых пользователь вручную правит INI‑файлы, а комментарии и оформление должны сохраняться.

Этот режим экспериментальный и **не рекомендуется для продакшена**.

### Creating a human‑editable reader

Вы можете открыть файл в human‑режиме синхронно или асинхронно:

```csharp
// Synchronous
NeoIniReader reader = NeoIniReader.CreateHumanMode("config.ini");

// Asynchronous
NeoIniReader readerAsync = await NeoIniReader.CreateHumanModeAsync("config.ini");
```

Когда human mode включён:

- Проверка checksum (`UseChecksum`) автоматически **отключается**, позволяя свободно редактировать файл вручную.
- Комментарии и форматирование **сохраняются и пере‑эмитятся** при сохранении.
- Режим **нельзя комбинировать с шифрованием** из соображений безопасности.

> **Warning:** Поскольку это экспериментальная фича, структура файла и поведение могут измениться в будущих релизах.
